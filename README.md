# EEG prediction using ML system

This project is a machine learning system, that is able to understand EEG signals from post-coma patients, it's purpose is to be able to generate a binary prediction
and help on real life medical decisions. 

## Phase 1: Initial pipeline

This phase is the beginning of the project and the goal is create an initial pipeline, with a few data segmentation to understand the nature and structure of EEG signals, and understand the models foundation. 

### Characteristics. 

* Dataset: ICARE v2.1
* Package manager:UV 
* Language: Python
